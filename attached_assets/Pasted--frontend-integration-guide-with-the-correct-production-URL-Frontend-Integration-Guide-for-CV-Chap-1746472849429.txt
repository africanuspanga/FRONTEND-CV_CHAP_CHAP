Frontend Integration Guide for CV Chap Chap
Production URL and Endpoints
The current production URL is:

https://cv-screener-africanuspanga.replit.app
Available Endpoints
Get All Templates

GET https://cv-screener-africanuspanga.replit.app/api/templates
Returns all available templates as JSON
No authentication required
Response includes template details: id, name, description, thumbnail_url, etc.
Preview Template with Data (Generate PDF without payment)

POST https://cv-screener-africanuspanga.replit.app/api/preview-template/{template_id}
Used for testing/previewing templates with CV data
Requires CV data in JSON format
Returns PDF directly or as Base64 (if X-Prefer-JSON-Response header used)
Download Test PDF (Pre-generated sample)

GET https://cv-screener-africanuspanga.replit.app/api/download-test-pdf/{template_id}
Returns a pre-generated sample PDF for the specified template
Good for testing download functionality
CV Creation Flow (with payment)

Initiate: POST https://cv-screener-africanuspanga.replit.app/api/cv-pdf/anonymous/initiate-ussd
Verify Payment: POST https://cv-screener-africanuspanga.replit.app/api/cv-pdf/{request_id}/verify
Check Status: GET https://cv-screener-africanuspanga.replit.app/api/cv-pdf/{request_id}/status
Download: GET https://cv-screener-africanuspanga.replit.app/api/cv-pdf/{request_id}/download

Example API Integration
Here's a sample JavaScript code to preview a template:

// Preview template with CV data
async function previewTemplate(templateId, cvData) {
  try {
    const response = await fetch(`https://cv-screener-africanuspanga.replit.app/api/preview-template/${templateId}`, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-Prefer-JSON-Response': '1'  // For base64 response
      },
      body: JSON.stringify(cvData)
    });
    
    if (response.ok) {
      const data = await response.json();
      if (data.success && data.pdf_base64) {
        // Create download link with base64 data
        const pdfBlob = base64ToBlob(data.pdf_base64, 'application/pdf');
        const pdfUrl = URL.createObjectURL(pdfBlob);
        downloadPdf(pdfUrl, data.filename || 'cv.pdf');
      } else {
        console.error('Failed to get PDF data:', data.error);
      }
    } else {
      console.error('API request failed:', response.status);
    }
  } catch (error) {
    console.error('Error previewing template:', error);
  }
}
