### **Pattern for ALL Other Templates**

Apply this same pattern to every template:

1. **Add Props interface with `colorOverride`**:
```tsx
interface Props {
  data: CVData;
  colorOverride?: string | null;
}
```

2. **Define DEFAULT_COLOR at top**:
```tsx
const DEFAULT_COLOR = '#XXXXXX'; // Template's original color
```

3. **Use colorOverride or default**:
```tsx
const primaryColor = colorOverride || DEFAULT_COLOR;
```

4. **Replace ALL hardcoded color values with `primaryColor`**:
```tsx
// Before:
style={{ color: '#0891B2' }}
style={{ backgroundColor: '#0891B2' }}
style={{ borderColor: '#0891B2' }}

// After:
```
