### 1. **Wrap the CV in a scaling container**

In your `ResumePreview.tsx` or similar file, wrap your existing CV template with this:

```jsx

import "./cvPreview.css"; // <- optional for cleaner CSS separation

export default function ResumePreview() {
  return (
    <div className="cv-preview-container">
      <div className="cv-scaler">
```

### 2. **Add this CSS (Tailwind or plain CSS)**

If you're using Tailwind (recommended):


```css
.cv-preview-container {
```
