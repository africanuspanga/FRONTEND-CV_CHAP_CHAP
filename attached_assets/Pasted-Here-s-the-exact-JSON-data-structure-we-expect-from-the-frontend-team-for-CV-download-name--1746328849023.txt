Required Fields
The following fields are required for all CV templates:

name
title
email
phone
location
summary
At least one entry in experience
At least one entry in education
At least four entries in skills
Optional Fields
The following fields are optional and will only be rendered if provided:

languages
certifications
projects
references
Field Limitations
name: Maximum 50 characters
title: Maximum 100 characters
summary: Maximum 500 characters
experience[].description: Maximum 500 characters per entry
education[].description: Maximum 300 characters per entry
skills: Maximum 15 skills
API Usage
The frontend team should:

Collect the user's CV data in this exact JSON format
Send a POST request to /download-pdf-test/{template_id} with the JSON in the request body
When the response is successful, redirect to the same URL with a GET request to download the generated PDF
